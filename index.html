Overview

forKernels is a comprehensive suite of high-performance Fortran libraries designed to deliver near-bare-metal speed for demanding computational workloads. Each module is optimized for modern HPC environments while providing easy integration through Python bindings.

Mission

To provide enterprise-grade, high-performance computing solutions that bridge the gap between raw computational power and practical application development.

Vision

Empowering developers and researchers with production-ready HPC libraries that scale from edge devices to cloud infrastructure.

Approach

Leveraging Fortran's proven performance with modern interfaces, enabling seamless integration into contemporary software ecosystems.

Library Suite

forMath

Comprehensive mathematical operations including linear algebra, matrix computations, and numerical methods optimized for scientific computing.

  • BLAS/LAPACK-level performance
  • Vector and matrix operations
  • Statistical functions

forAI

AI and machine learning primitives designed for training and inference workloads with focus on neural network operations.

  • Neural network kernels
  • Activation functions
  • Optimization algorithms

forDSP

Digital signal processing kernels for filtering, transforms, and real-time signal analysis applications.

  • FFT/IFFT operations
  • FIR/IIR filters
  • Spectral analysis

for3D

3D graphics and geometry operations for rendering, computer vision, and spatial computing applications.

  • Matrix transformations
  • Geometric primitives
  • Collision detection

forCUDA

GPU acceleration interfaces providing seamless access to CUDA-enabled hardware for parallel processing.

  • GPU memory management
  • Kernel execution
  • Multi-GPU support

forQuant

Quantitative computing and financial mathematics for risk analysis, pricing models, and portfolio optimization.

  • Risk analytics
  • Monte Carlo simulation
  • Time series analysis

Getting Started

forKernels libraries are designed for easy integration into your existing workflows. Choose your preferred installation method below.

Python Installation

pip install formath forai fordsp

Install individual libraries via pip for immediate Python integration.

Container Deployment

docker pull ghcr.io/forkernels/forai:latest

Use pre-built containers for rapid deployment in cloud environments.

Source Build

git clone https://github.com/forKernels/forMath
cd forMath && make install

Build from source for custom configurations and optimizations.
